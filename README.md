# Üretim Takip Sistemi (Product Tracking System)

Bu proje, bir fabrikadaki üretim ve duruş sürelerini takip etmek ve standart molaları (Çay, Yemek) otomatik hesaplamak için geliştirilmiş bir ASP.NET Core MVC uygulamasıdır.

## Özellikler
* **Dinamik Tablo:** Kullanıcılar üretim verilerini dinamik olarak ekleyip silebilir.
* **Otomatik Hesaplama:** Girilen üretim aralıklarına denk gelen standart molalar (10:00, 12:00, 15:00) algoritma tarafından otomatik tespit edilir ve araya eklenir.
* **Eksik Mola Kontrolü:** Üretim yapılmayan saatlerdeki molalar da raporda gösterilir.
* **Teknolojiler:** ASP.NET Core MVC, C#, JavaScript, Flatpickr (Tarih Seçici).
